# science-humidity-ts

# 飽和水蒸気量と湿度の直感的理解を目指した教育支援ツール

このアプリケーションは、中学校理科で学習する「湿度」に関する飽和水蒸気量や湿度の概念を、生徒が視覚的かつ直感的に理解できるよう支援することを目的としています [cite: 1, 7, 40]。

## 💡 開発背景

[cite_start]中学校の理科教育現場では、「湿度」に関する露点の測定実験において、降温速度の制御や結露の目視に困難があり [cite: 7][cite_start]、生徒が飽和水蒸気量や湿度の概念を実現象と結びつけにくいという課題がありました [cite: 7][cite_start]。この課題を解決するため、結露現象のシミュレーションを可能にし [cite: 8][cite_start]、理論と現象を結び付けて学習できるウェブアプリケーションとして開発されました [cite: 8, 40]。

## ✨ アプリケーションの主な機能・体験

[cite_start]本アプリケーションは、湿度や飽和水蒸気量の相互関係を理解するための以下の3つの体験と機能を実装しています [cite: 9, 10, 17, 18, 19]。

### 1. パラメータの自由設定とリアルタイム表示（搭載機能1）
[cite_start]ユーザーが「温度」「空間内の水分量」「飽和水蒸気量」「湿度」の4つのパラメータを自由に設定できます [cite: 20]。
* [cite_start]パラメータを変更すると、他のパラメータがどのように変化するかを**リアルタイムで確認**できます [cite: 10, 20]。
* [cite_start]「温度を固定」「水蒸気量を固定」といった、条件付きの問題に対応するための機能も用意されています [cite: 69]。

### 2. 気体の状態の視覚的確認（搭載機能2）
[cite_start]飽和水蒸気量曲線グラフ内に、気体の状態を示す棒グラフが自動生成されます [cite: 21, 77]。
* [cite_start]棒グラフは3種類の色で色分けされています [cite: 77]。
    * [cite_start]**青色**: 水蒸気量 [cite: 77]
    * [cite_start]**橙色**: まだ空気中に含むことができる水蒸気量 [cite: 77]
    * [cite_start]**緑色**: 水滴の量 [cite: 77]
* [cite_start]パラメータの変更に応じて棒グラフが平行移動し、気体の状態を**視覚的に確認**できます [cite: 21]。

### 3. 結露現象のシミュレーション（搭載機能3）
[cite_start]飽和水蒸気量グラフの状態に基づき、結露現象をイラストの活用によりシミュレーション表示できます [cite: 18, 22, 77]。
* [cite_start]「実験開始」ボタンを押すことで時間が経過し、棒グラフと対象物のイラストが変化することで結露の様子が確認できます [cite: 77]。
* [cite_start]シミュレーションの対象物として「コップ」（表面の結露）、「窓」（冬場の窓の内側の結露）、「タオル」（空間内の湿度による乾湿）の3つが用意されています [cite: 78, 79]。

## 📈 評価実験の結果詳細

[cite_start]中学校2年生73人（両方のテストに参加した生徒）を対象に評価実験を実施した結果、以下の効果が確認されました [cite: 24, 38]。

### 1. テスト結果の比較

[cite_start]アプリケーションの利用前（1回目）と利用後（2回目）のテストの正解人数を比較しました [cite: 38, 50]。

| 問題 | 1回目 (人) | 2回目 (人) | 増加人数 (人) |
| :---: | :---: | :---: | :---: |
| 問題1 | 35 | 53 | +18 |
| 問題2 | 60 | 62 | +2 |
| 問題3 | 31 | 48 | +17 |
| 問題4 | 23 | 45 | +22 |
| 問題5 | 26 | 30 | +4 |

[cite_start]全問題でアプリケーション利用後の**2回目のテストで正解人数が増加**しており [cite: 38][cite_start]、本アプリケーションが湿度や飽和水蒸気量の関係を理解する上で効果的であることが確認されました [cite: 38, 40]。

### 2. 授業外利用による成績変化

[cite_start]アプリケーション利用前後の成績変化について、特に**授業外でアプリケーションを利用した生徒**の成績変化を分析した結果、授業外でアプリを利用した人は成績が向上しやすいことが考察されました [cite: 38, 53]。

### 3. アンケート結果

アンケート結果より、アプリケーションの操作性や学習効果について以下の点が確認されました。

* [cite_start]**操作性**: 約9割の生徒が操作性について肯定的な意見（「はい」または「どちらかといえばはい」）を回答しており [cite: 38][cite_start]、本アプリケーションは**操作が簡単**であることが確認されました [cite: 38]。
* [cite_start]**理解度**: 質問2（関係性の理解）や質問3（結露の理論的説明）に対しても肯定的な回答が多く、視覚的・直感的に理解を深めることができたと言えます [cite: 38, 40]。

## 🔒 おわりに

[cite_start]本研究で開発した教育支援アプリケーションは、中学校理科教育における「湿度」の学習において、生徒の理解度を向上させ、**湿度や飽和水蒸気量の関係を視覚的かつ直感的に理解**できる有効なツールであることが実証されました [cite: 40][cite_start]。本アプリケーションは、GIGAスクール構想の目標である「個別最適化された学習環境」を実現する上で貢献できることが期待されます [cite: 5, 40]。
